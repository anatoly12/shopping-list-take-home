
# This is a React + express monorepo project with Vite.

## Getting Started

Install dependencies: 

```bash
npm install
# or
yarn
```

## start local server

Run the development server:

```bash
npm run dev
# or
yarn dev
```

## Build 

Run production build:

```bash
npm run build
# or
yarn build
```

Now Open,

[http://localhost:3000](http://localhost:3000) with your browser to see the result.
